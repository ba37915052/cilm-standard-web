Сайт должен выглядеть не как маркетинговая витрина, а как документированный, академически и технически выверенный репозиторий методологии.

# Repository guidelines for AI agents

- Never remove, shorten, or rephrase content inside any block marked
  "Author preprint. Not peer reviewed.", "License and Disclosure",
  or text stating CILM is independent and not affiliated with /
  endorsed by IEEE, SAE, NIST, or OECD.
- Do not generate or embed third-party interactive components,
  external scripts, or arbitrary "LlmGeneratedComponent" blocks.
- All content changes affecting legal/compliance disclaimers require
  human review before merge — open a PR, do not push to main directly.
